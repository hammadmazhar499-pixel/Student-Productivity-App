## Project Description

The Student Productivity App is a web-based application designed to help students manage their academic tasks efficiently. The application provides an easy-to-use interface for organizing and tracking student-related activities and productivity tasks.

---

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* JSON Server

---

## Installation

Before running the project, make sure that Node.js is installed on your system.

Install the required dependencies by running:

```bash
npm install
```

---

## Running the Application

### Step 1: Start the JSON Server

Run the following command in the project directory:

```bash
npm run server
```

This will start the JSON Server and make the data available to the application.

### Step 2: Open the Application

After the server is running, open the `index.html` file in your web browser.
